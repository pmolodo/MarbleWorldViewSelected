v1.1.0
------
- Focus position uses xform gizmo, not rendering-bounding-box
  - rendering bounding box was often non-intuitive
  - works with multiple objects selected
  - changes when you change snap points with tab
- Include uninstall script
- Improved documentation
- Numerous build tweaks

v1.0.0
------
- Initial release of plugin.
- Adds two features:
  - press v to move the camera to view the selected object
  - drag with middle-mouse to orbit the view around the selected object (ie, look at it from different angles, keeping it centered in the view)
- See README.md for installation instructions.
